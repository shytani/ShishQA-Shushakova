DNS — система доменных имен, которая обеспечивает связь между наименованием сайта и его цифровым адресом.

HTTP Протокол Передачи Гипертекста. 

HTTPS - это защищенный протокол передачи гипертекста. 
Обеспечивает зашифрованную и безопасную идентификацию сетевого сервера.

Разница:

HTTP					
•	Hypertext Transfer Protocol 		
•	Порт по умолчанию – 80 		
•	http://				
•	Нет защиты передаваемой 
информации. 
•	Не использует шифрование
•	Сайтам с HTTP не нужен SSL	

HTTPS
•	Hypertext Transfer Protocol Secure
•	Порт по умолчанию -  443
•	https://
•	HTTPS шифрует данные перед передачей их по сети. 
•	Для работы с HTTPS нужен SSL сертификат

Разница версий
В HTTP 1.0 соединение закрывается после одной пары запрос/ответ.
В HTTP/1.1 введён режим «постоянного соединения». 
HTTP/2 в отличие от предыдущих версий, является бинарным вместо текстового, 
передача нескольких асинхронных HTTP-запросов по одному TCP-соединению,  
сжатие заголовков методом HPACK, несколько ответов на один запрос.
